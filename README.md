## Features Implemented
- ✅ Homepage with store name and welcome message
- ✅ Product pages with minimum 5 products (images, prices, descriptions)
- ✅ About Us page with store description and mission
- ✅ Contact page with email, phone, and contact form
- ✅ Cart functionality (Add to Cart — built-in Shopify feature)
- ✅ Mobile-responsive design
- ✅ Store navigation menu

## Challenges
- Setting up the store theme and customizing it to match the brand identity
- Managing duplicate pages and cleaning up the store structure
- Understanding how Shopify's built-in cart and product pages work
- Accessing the live store preview during the trial period

---

## Lessons Learned
- How to use a No-Code platform (Shopify) to build a fully functional e-commerce store
- How to create and manage products, pages, and navigation in Shopify
- The importance of clean store structure and removing duplicate content
- How Shopify handles cart functionality automatically without custom code
- How to customize a storefront using the Shopify Theme Editor
